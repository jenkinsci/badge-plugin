Adds one or more error badges to the current Pipeline build. Error badges text messages with an icon and an optional clickable hyperlink. Error badges are often used to provide a visual indicator of additional status information about a Pipeline build. Error badges added by addErrorBadge are displayed in the build history of the Pipeline job. Error badges that either consist of only an icon and a toolip text or only text will also have a copy to clipboard feature enabled by default.

Example: Badge with text

The following example adds an error badge with the text, "This is alarming": 

addErrorBadge text: 'This is alarming'

Example: Badge with text and link

The following example adds an error badge with the text, "This is alarming" and a link: 

addErrorBadge text: 'This is alarming', link: 'https://issues.jenkins.io/browse/JENKINS-59646', target: '_blank'
